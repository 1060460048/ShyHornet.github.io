Jkeyll初探

front-end html css javascript Jquery Jkeyll

15 Oct 2015

昨天建立了自己的Jkeyll博客,研究了一整天Jkeyll的工作原理和如何进行配置。

首先是安装

这一步很简单,只需在终端输入:

$ sudo gem install jekyll

创建一个简单的博客并运行:

$ jekyll new helloJekyll
$ cd helloJkeyll
$ jekyll serve

在浏览器中输入localhost:4000即可观看效果:
